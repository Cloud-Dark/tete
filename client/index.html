Upload Endpoints

POST/api
Upload single file
curl -F "file=@myfile.txt" /api
POST/api/upload
Upload multiple files
curl -F "files=@file1.txt" -F "files=@file2.txt" /api/upload
POST/api/text
Upload text content
curl -H "Content-Type: application/json" -d '{"text": "Hello", "filename": "test.txt"}' /api/text

Download & Info

GET/file/:id
Get file information
curl /file/abc123
GET/file/:id/download
Direct download
curl -O /file/abc123/download
DELETE/file/:id
Delete file
curl -X DELETE /file/abc123
